# 歡迎來到Qwik

Qwik 是一種新型 Web 框架，可以提供任何大小或複雜性的 Web 應用程式即時載入。您的網站和應用程式可以使用大約 1kb 的 Javascript（無論應用程式複雜性如何）啟動，並在規模上實現一致的性能。

[查看更多介紹、影音以及Podcast](/media/)

## Qwik 的目標

<div class="card-grid">
  <div class="card">
    <h3>一般用途</h3>
    <p>Qwik 可用於構建任何類型的網站或應用程式</p>
  </div>
    <div class="card">
    <h3>即時啟動</h3>
    <p>與其他框架不同的是，Qwik是可恢復的(resumable)，這意味著Qwik應用程式具備零水合化( 0 hydration )。這使得Qwik應用程式無論其大小或複雜性如何，都具有即時的互動性。</p>
  </div>
    <div class="card">
    <h3>速度優化</h3>
    <p>Qwik具有前所未有的性能，即使在行動設備上也能提供亞秒級( sub-second )的全頁面載入。 Qwik通過提供純HTML，並在需要時逐步載入Javascript來實現這一目標。</p>
  </div>
</div>

## 頁面速度真的重要嗎??

簡而言之：速度慢的網站讓訪客望而卻步，使企業造成損失。快速的網站有更好的搜尋引擎，以及更好的用戶體驗，並且更有利可圖。

一些案例來自 [web.dev](https://web.dev/tags/case-study/):

|                                                                                                                                                                                                                                               |                                                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **每增快100毫秒→增加1%的轉換量** <br /> 對於Mobify來說，首頁載入速度每降低100毫秒，基於頁面訪問的轉換率就會增加1.11%，產生平均每年近38萬美元的收入增長。             | **速度提高50%→銷售量增加12%** <br /> 當AutoAnything將頁面載入時間減少一半時，他們的銷售額提高了12%到13%。                      |
| **速度提高20% → 轉換率提高10%** <br /> 零售商 Furniture Village 審視了他們的網站速度，並制定了一個計劃來解決他們發現的問題，使得頁面載入時間減少20%，轉換率增加10%。 | **速度提高40% → 簽約人數增加15%** <br /> Pinterest將預期的等待時間減少了40%，這使搜尋引擎的流量和註冊人數增加了15%。  |
| **提升速度850毫秒 → 增加7%的轉換率** <br /> COOK將平均頁面載入時間減少了850毫秒，使轉換率提高了7%，跳出率降低了7%，並且每個頁面的訪問率增加了 10%。                    | **延遲1秒→減少10%的用戶** <br /> 英國廣播公司(BBC)發現，他們的網站每多花一秒鐘載入，就會多損失10%的用戶。 |
|                                                                                                                                                                                                                                               |                                                                                                                                                        |
